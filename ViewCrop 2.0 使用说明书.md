## 问题诊断

### 问题现象
1. EXE 文件正常生成（大小 7,365,099 字节）
2. 启动后进程在任务管理器中可见
3. **但没有 GUI 窗口显示**

### 可能原因
1. **Qt 插件路径问题**: PySide6 需要正确的 platform 插件路径
2. **初始化阻塞**: main_window.py 中某些代码阻塞了 GUI 线程
3. **样式/字体问题**: 某些样式设置导致窗口不可见

### 已尝试的修复
✅ 已添加 PyInstaller 兼容性代码
✅ 已设置 QT_QPA_PLATFORM_PLUGIN_PATH
✅ 已禁用 UPX 压缩
✅ 已包含所有隐藏导入
✅ 已检查 qwindows.dll 存在

### 推荐解决方案
1. **临时方案**: 直接使用源码模式（`快速启动.bat`）
2. **调试方案**: 
   - 运行 `运行诊断.bat` 获取详细日志
   - 检查 Windows 事件查看器
   - 使用 Process Monitor 监控文件访问
3. **长期方案**: 
   - 重新设计 main.py 初始化流程
   - 添加更多调试日志
   - 尝试不同的打包配置

## 功能说明

### 新增功能

1. **微调按钮 (+/-)**
   - 每个参数滑块旁有 +/- 按钮
   - 精确调整参数值

2. **区域数量显示器**
   - 大号字体显示检测到的区域数量
   - 颜色提示：红(0)/橙(1-4)/绿(5-19)/蓝(20+)

3. **拖拽支持**
   - 支持拖拽图片文件到窗口
   - 自动加载并处理

4. **单视图模式**
   - 只显示结果视图（带绿色边框）
   - 界面更简洁
