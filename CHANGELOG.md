# MultiViewCropping 更新日志

## [2.1.0] - 2026-03-16

### 新增功能 🎉

#### 二值化掩膜预览（Binary Mask Preview）
- **新增主界面预览模式切换**，在调参时即时查看二值化掩膜（黑底白块）
- 位于主界面预览区顶部的两个按钮切换预览模式

#### 预览模式
- 📷 **最终结果**：带绿色边框标注的处理结果（默认模式）
- 🔲 **二值化掩膜**：黑底白块的二值化掩膜（Binary Mask）
  - 白色区域 = 检测到的前景对象（模型/手/衣服）
  - 黑色区域 = 背景
  - 基于膨胀后的轮廓生成

#### 使用方法
1. 拖入或选择图像文件
2. 点击预览区顶部的 **"🔲 二值化掩膜"** 按钮查看掩膜
3. 调整右侧参数（融合度、过滤阈值等）
4. 预览区实时更新，显示当前掩膜状态

#### 技术细节
- 掩膜由 `cv2.findContours` 和形态学膨胀生成
- 白色连通域对应绿色边框的检测区域
- 调参时掩膜实时更新，便于优化检测效果
