# ViewCrop 2.0 - 智能图像分割工具

> 经典重构版本，简洁、稳定、高效

## 🌟 特性

- **智能分割** - 自动识别图像中的独立区域
- **实时预览** - 参数调整即时显示效果
- **批量处理** - 支持多张图片同时处理
- **高质量导出** - 保持原始图像质量
- **简洁界面** - 专注核心功能，操作直观

## 📋 环境要求

- Python 3.8+
- OpenCV 4.8+
- NumPy 1.24+
- PySide6 6.5+
- Pillow 9.0+

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

### 3. 基本使用

1. **添加图片** - 点击"添加文件"或直接拖拽图片到窗口
2. **调整参数** - 使用右侧滑块调整融合度和过滤阈值
3. **查看结果** - 左侧显示原图和处理结果对比
4. **导出图片** - 点击"导出结果"保存分割后的图片

## 🎛️ 参数说明

| 参数 | 作用 | 使用场景 |
|------|------|----------|
| 融合度 | 控制相邻区域合并程度 | 值越大，越容易合并相邻小块 |
| 过滤阈值 | 最小区域面积过滤 | 剔除太小的杂点区域 |
| 移除水平线 | 去除图像中的水平边缘线 | 有表格、横线干扰时使用 |
| 移除垂直线 | 去除图像中的垂直边缘线 | 有竖线、边框干扰时使用 |

## 🔧 核心算法

### 图像处理流程

1. **灰度转换** - 将彩色图像转为灰度
2. **边缘检测** - 使用Canny算法提取边缘
3. **线段过滤** - 可选移除水平/垂直线
4. **图像膨胀** - 连接相邻的边缘区域
5. **轮廓检测** - 查找独立区域边界
6. **区域过滤** - 按面积过滤小区域
7. **结果标注** - 在图像上绘制边界框和编号

### 技术特点

- **高效算法** - 优化的OpenCV处理流程
- **内存友好** - 智能缓存机制
- **并行处理** - 支持多线程处理（规划中）
- **容错机制** - 完善的异常处理

## 📊 性能指标

## 🐛 常见问题

### Q: 程序无法启动？
A: 检查Python版本和依赖包是否正确安装

### Q: 中文文件名乱码？
A: 确保使用UTF-8编码，程序已处理中文路径

### Q: 处理结果不理想？
A: 尝试调整参数：
- 增大融合度连接分离的区域
- 增大过滤阈值去除小碎片
- 启用线段过滤去除干扰

### Q: 导出的图片质量低？
A: 内置导出保持原图质量，检查是否被压缩
