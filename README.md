# ViewCrop 2.0 - 智能多视图批量裁切工具

> 经典重构版本，简洁、稳定、高效

## 🌟 简介

通过调节**融合度**与**过滤阈值**双参数，实现批量多视图智能裁切，自动消除不同图片的干扰噪声。

## 🎯 核心功能

| 参数 | 解决什么问题 |
|------|------------|
| **融合度** | 控制相邻区域合并程度 —— 应对同一对象被割裂的情况 |
| **过滤阈值** | 最小区域面积过滤 —— 剔除小杂点、噪点干扰 |
| **移除水平线** | 去除表格线、横线等结构性干扰 |
| **移除垂直线** | 去除边框、竖线等结构性干扰 |

## 🚀 快速开始

```bash
pip install -r requirements.txt
python main.py
```

## 📋 环境要求

- Python 3.8+
- OpenCV 4.8+
- NumPy 1.24+
- PySide6 6.5+
