## 💡 使用教程

### 第一步：添加图片

**方法 A - 拖拽**
1. 打开 Windows 资源管理器
2. 选择一张或多张图片
3. 拖拽到程序窗口任意位置
4. 程序自动加载并显示

**方法 B - 按钮**
1. 点击"添加文件"按钮
2. 选择要处理的图片
3. 确认后添加到列表

### 第二步：调节参数

**融合度** (默认值)
- 范围：0-50
- 作用：控制边缘连接强度
- 建议：从默认值开始微调

**过滤阈值** (默认值)
- 范围：100-5000
- 作用：过滤掉小面积区域
- 建议：太大会丢失小区域，太小会有噪声

**水平/垂直过滤**
- 勾选后可移除对应方向的边缘
- 组合使用可过滤矩形框

### 第三步：观察结果

左侧大窗格显示带绿色边框的标注结果：
- 绿色边框 = 检测到的有效区域
- 数量显示器 = 当前检测数量及状态
- 实时反馈 = 调节参数立即更新

### 第四步：导出结果

1. 确保图片和参数都满意
2. 点击"导出结果"按钮
3. 选择保存目录
4. 按文件名自动分类保存

## 🎯 最佳实践

### 场景 1: 检测电路板元件

```
融合度：15-25
过滤阈值：500-1000
水平过滤：✓
垂直过滤：✓
```

**理由**: 中等融合度保证连通性，过滤阈值适中，双向过滤去除引脚干扰

### 场景 2: 检测文字区域

```
融合度：30-40
过滤阈值：200-500
水平过滤：✗
垂直过滤：✗
```

**理由**: 高融合度连接笔画，低过滤阈值保留小文字

### 场景 3: 检测产品缺陷

```
融合度：5-15
过滤阈值：1000-2000
水平过滤：✗
垂直过滤：✗
```

**理由**: 低融合度保持细节，高过滤阈值忽略微小瑕疵

## 🐛 已知问题

### 问题 1: 首次启动较慢
**原因**: PyInstaller 打包的 exe 首次运行需要解压临时文件  
**解决**: 等待 2-3 秒，后续启动正常

### 问题 2: 超大图片 (>4K) 处理慢
**原因**: 内存占用高，CPU 密集计算  
**解决**: 建议先缩放图片至 2K 以内再处理

### 问题 3: 某些特殊格式图片不支持
**支持的格式**: PNG, JPG, JPEG, WEBP  
**原因**: OpenCV 限制
